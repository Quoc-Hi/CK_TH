EAS Build Preview Links for Movie Watchlist
=============================================

Để tạo preview build, chạy các lệnh sau:

Android:
  eas build --profile preview --platform android

iOS:
  eas build --profile preview --platform ios

Sau khi build hoàn thành, link preview sẽ được hiển thị trong terminal hoặc trên Expo dashboard.
